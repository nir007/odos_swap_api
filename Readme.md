# Swap eth, bnb, matic, usdt, usdc by ODOS API 

### Support chains 
 - arbitrum
 - bsc
 - scroll
 - optimism
 - polygon

### .env file example

````
PROXY=login:passwod@IP:PORT
PRIVATE=your_wallet_private_key
BASE_URL=https://api.odos.xyz
````
